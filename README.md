# Valdo
### Your personal healthcare assistant

With Valdo, we aim to increase efficiency on logging daily health data and provide personal feedbacks based on the data.  
Valdo is powered by Meta's Llama 3.1-8B and OpenAI's GPT 5 Mini to ensure **speed** and **accuracy**.

### Overview
Valdo is a web with straightforward features to boost user's experience.  
Its features are simple, helpful, and **personal**.  
Only features essential to develop a healthy lifestyle and "prevent" sickness are included to prevent overwhelming users.

### Key Features
- Llama 3.1-8B powered AI health coach.
- GPT 5 Mini powered meal logger
- User daily status recording system.
- Llama 3.1-8b powered user status analyser.

### AI Tools
